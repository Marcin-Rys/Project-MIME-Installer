# Project-MIME-Installer
An installer for whole MIME project. 

## Installation
1. Clone this repository 
```
bash
git clone https://github.com/Marcin-Rys/Project-MIME-Installer.git
cd Project_MIME_Installer
```
2. Run installation script
```
bash '.\install.sh'
```
3. Go to folder Clown_Project and edit your variables in .env
